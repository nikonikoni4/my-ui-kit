# TaskCalendar 测试页面

## 📍 位置
`test/src/test_component/TaskCalendarTest.tsx`

## 🎯 功能说明

这是一个完整的 TaskCalendar 组件测试页面，包含以下功能：

### 1. 日历展示
- ✅ 月视图和周视图切换
- ✅ 左右箭头导航（切换月份/周）
- ✅ 今天标记
- ✅ 磨砂玻璃设计风格

### 2. 任务管理
- ✅ 显示预设的模拟任务数据
- ✅ 点击任务查看详情（记录到日志）
- ✅ 点击空白日期添加新任务
- ✅ 任务按日期分组显示

### 3. 交互功能
- ✅ 日期选择（点击日期单元格）
- ✅ 选中日期信息显示
- ✅ 操作日志记录（最近10条）
- ✅ 实时任务统计

### 4. 信息面板
- **选中日期信息**：显示当前选中日期的详细信息和任务数量
- **操作日志**：记录所有用户交互操作，带时间戳
- **功能说明**：提供组件使用指南

## 🚀 如何访问

1. 启动开发服务器：
   ```bash
   cd test
   npm run dev
   ```

2. 在浏览器中打开应用（通常是 `http://localhost:5173`）

3. 点击顶部的 **TaskCalendar** 标签页
